# Проект Прогноз Погоды 
 
Данный проект написан на языке JavaScript с использованием React на базовых функциях.  
Данные погоды, получают из внешнешних источников через fetch. 
 
Выбор одежды реализован на условиях if\else а также условия мужчина или женщина смотрит 
Также динамически меняется погода в зависимости от какие погодные условия на улице. 
 
В погоде используются 5 городов России так как на бесплатной версии можно использовать 5 городов только. 
 
Приложение адаптировано под мобильные устройства и планшеты.
